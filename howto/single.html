To get single attributes you need to parse the JSON string. Let's take a look at what this looks like.

Attributes listed here include country name, capital, alternative spellings, relavance, region, etc

You an see that name, capital, and relevance are followed by only one attribute, whereas alternative spellings (altSpellings) has 2. In this section, we will focus on just getting those single attributes. You can learn how to get multiple attributes in the next section

1. Create a reqeust object

var xhr = new XMLHttpRequest();

This request object can have the methods open and send enacted upon it. These methods allow data transfer between your website and the server.

2. Enact JSON parsing on your XML object once page has loaded

xhr.onload = function(); responseObject = JSON.parse(xhr.responseText);

The onload method ensures that whatever is inside your function will be enacted once the page has loaded.

This calls the responseText method on your object, which converts the data in the object into a text. This text has to be parsed into value for the JSON array.

The variable responseObject stores this data.

3. Enact the get and open method on your XML object

xhr.open('GET', 'http://restcountries.eu/rest/v1/all', true); xhr.send();

The first statement gets the data from your requested url and the second sends it to the server

4. Create content

var newContent = responseObject[i]["**attribute name**"]

ie responseObject[i]['name']

The JSON content is delivered in array format. Each index of the array contains information about that country. To derive informaiton about that country's name, you have to specify what you want after the iterator -- in this case name ([i][name])

5. Stable content onto page

document.getElementById('content').innerHTML = newContent;

This finds the content id (getElementById) on the document html(innerHtML) object and replaces it with the variable you created
