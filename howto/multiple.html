Multiple Attributes

By now we've looked at how to get single attributes. This section will look into multiple attributes. Borders, for example fill this criteria.

Good News!

Good news is that you've done most of the work in term of creating the XML object and we will use this as a foundation for obtaining multiple objects.

You can get the whole list of multiple objects in one attribute by treating it as a single attribute. The below gives all the borders of a country

ie responseObject[i]['borders']

Specificity

However, if you want a specific border, you will need to specify which one with the syntax below

responseObject[i]['borders'][0]

This will give you the first border of each country if there is one. If not, you'll get an undefined object. You can make a conditional statement to only give you countries with more than X number of border.

use responseObject[i]['borders'].length to specify the delimter
