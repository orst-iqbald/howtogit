The Services

See the picture below for some of the services provided by this API. This means that you can obtain these via form input or you can specify your URL to this.

For example, your form can have a name selector field, which can select data from a particular country.

Write in your country to pick

The code used to create this is

    var url = 'http://restcountries.eu/rest/v1/name';
    var finalurl = url + '/' + country;

To get this API request, I ultimately had to have this url for my JSON string: http://restcountries.eu/rest/v1/name/country

You can see that finalurl combines this by obtaining the country from the user
